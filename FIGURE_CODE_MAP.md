## Delete / do not publish as source code

- `test.py`: only a local debug loader.
- `__pycache__/`: compiled Python cache, not source.
- every `*.Zone.Identifier`: Windows metadata, not source.
- random notes inside old run folders, for example `gghll.txt`, `hjjjj.txt`, `fontzzz.txt`, etc.
- generated run folders such as `runs/`, `run_strategy2/`, `run_strategy3/`, `run_strategy4/`, unless you intentionally want to publish raw outputs/NPZ data.
- intermediate draft images `figure7.png` and `figure7_padded_same_aspect.png` if the manuscript uses `figure7_equal_size_to_figure8.png`.

## Important reproducibility warning

For a clean GitHub repository, source code alone is not enough to regenerate every final PNG exactly as shown in `draft12` because several draft images are post-processed crops of raw script outputs, and Fig. 13 needs external reference CSV files that were not included in the uploaded ZIP. If the repository claims exact reproducibility, add either:

1. the final manuscript figures under `figures/`, plus this map, or
2. small post-processing scripts that crop/rename the raw outputs into the exact manuscript filenames.
