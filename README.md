# Patchless MF-VPINN Numerical Experiments

This repository contains the Python codes used for the numerical experiments of the Patchless MF-VPINN method.

The codes implement the main numerical experiments, adaptive enrichment strategies, error evaluation, and plotting routines used to generate the reported results. They are written to reproduce and check the computational results related to the manuscript figures.

The scripts include implementations for different refinement strategies, numerical error tracking, indicator-based enrichment, and post-processing of the generated outputs.

The required Python packages can be installed using:

```bash
pip install -r requirements_minimal.txt
```

These codes are provided to make the numerical part of the work easier to inspect, run, and verify.
